# PUSHER POLL

The program surveys user their preferred OS system and shows answer distribution.

### Prerequisites

A Pusher account and a newly created project

## Deployment

Currently has not been deployed to public domains.

## Built With

* NodeJS
* ExpressJS
* CanvasJS
* PushJS
